痕跡を、残さない。

Cookie・キャッシュ・Web ストレージ非永続

Cookie・キャッシュ・Web ストレージはセッション間で残しません。閲覧履歴は端末内のみに保存し、外部には一切送信しません。

覗かれても、見えない。

バックグラウンドで自動カモフラージュ

アプリをバックグラウンドに移した瞬間、画面は自動でカモフラージュされます。誰かの視線が向いた瞬間に、反射的に守れます。

一瞬で、消せる。

ワンタップ消去で一括削除

アドレスバー横の赤シールドを押すと、現モードのタブ・最近閉じたタブ・Cookie・キャッシュ・履歴を範囲指定で一気に消せます。

自分のプライバシーは、
自分で守る。

内緒モードで閲覧領域を分離

Face ID で保護される内緒モード。ブックマーク・履歴・タブを通常領域とは別に持てます。Focus フィルターから自動で切替も可能です。

広告もトラッカーも、遮断。

2 段階ブロック + 指紋対策

ネットワーク層と DOM 層の 2 段階で広告・トラッカーを遮断。Canvas / WebGL / AudioContext / Navigator への指紋採取スクリプトも内蔵で抑制します。
